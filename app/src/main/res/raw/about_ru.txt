Приложение предназначено для считывания данных темепературы, зафиксированных температурным логгером NHS3100.
Для полноценной работы приложения Android-устройство должно иметь NFC-модуль.
Для считывания данных с температурного логгера приложите устройство к датчику. (Проверьте, включен ли NFC-модуль!)
При успешном считываний температуры, на странице "Температура" отобразится информация о состоянии температурного логгера и текущая температура, котороую он считал в данный момент.
Для считывания большого количества данных необходимо подержать устройство рядом с датчиком некоторое время. Количество считанных значений отобразится в виде сообщения.
После успешного считывания данных вкладку "Температура" можно свайпнуть вправо, где на второй вкладке страницы будет доступна информация о текущей конфигурации логгера, особенностях считанных значений.
На третьей вкаладке страницы выводится график изменения температуры во время измерений. Считанные данные можно скачать в формате CSV в папку Downloads. (Проверьте разрешения приложения на использование файлов и медиаконтента!).
Считанная информация добавляется на страницу "История". При выборе из списка записи, открывается окно содержащее три вкладки, содержащие записи по каждому обмену данными с темепературным логгером.
На странице "Настройки" можно задать парметры работы температурного логгера:
- Время до начала измерений;
- Периодичность измерений;
- Продолжительность измерений;
- Диапазон температур.
При нажатии на кнопку "Задать параметры" Android-устройство перейдет в режим обмена информацией с температурным логгером для передачи параметров. По окончании передачи будет выведено сообщение о результате передачи.
При нажатии на кнопку "Сброс" и соединения с температурным логгером, настройки последнего очистятся и все сохраненные данные тмпературы будут удалены.
В правом верхнем углу приложения есть кнопка настройки. Которая позволяет очистить историю и выбрать язык приложения (Русский или Английский).
